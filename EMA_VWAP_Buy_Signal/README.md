# EMA-VWAP Crossover Buy Signal

## What the Indicator Does
This Pine Script detects buy signals when the EMA crosses above the VWAP, indicating potential upward momentum.

## Best Use Cases
- Ideal for day traders targeting momentum shifts.
- Use on 5-15 minute charts for stocks or forex.
